## Key Learnings

### What is an Embedded System?
- An embedded system is a computer designed for a specific task.
- It combines hardware and software.
- Examples include washing machines, cars, ATMs and medical devices.

### What is a Computer Program?
- A computer program is a sequence of instructions written to perform a task.
- Computers understand machine language, so C programs must be translated before execution.

### C Program Standardisation
- C was standardized so programs behave consistently across different systems.
- Standardization improves portability and compatibility.

### File Generation Sequence
- A C source file (.c) is processed by the compiler.
- The program goes through compilation/linking before becoming an executable.
- The executable can then run on the target machine.

### Host Machine vs Target Machine
- Host Machine: The computer where the program is written, compiled and debugged.
- Target Machine: The device where the final program executes.
